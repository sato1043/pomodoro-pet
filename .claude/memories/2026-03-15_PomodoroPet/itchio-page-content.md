Pomodoro Pet is a 3D virtual pet Pomodoro timer for Windows.

A cute wild boar walks alongside you while you work, helping you stay focused and take proper breaks. Watch as the environment changes with real-time weather, day/night cycles, and the 72 Japanese microseasons (Shichijuni-kou).

Features:
- Pomodoro Timer — Work / Break / Long Break cycling with circular progress ring
- 3D Character — Autonomous behavior, click/pet/drag interactions, emotion system
- Fureai Mode — Feed the character, build affinity over time
- Weather & Environment — 3 scene presets (meadow / seaside / park), 4 weather types, astronomy-based day/night cycle
- 72 Microseasons — Japanese traditional seasons displayed in real time
- World Map — Set your location, see local time and climate
- Statistics — 13-week heatmap, daily summaries, emotion trend charts
- Sound — Procedural ambient sounds (forest / rain / wind), timer SFX, break BGM
- Background Notifications — System notifications when the app is in the background
- Auto Update — Seamless updates via the app

Free Trial: All features available for 30 days. After the trial, the basic timer and character remain free. Register with your download key to unlock all features permanently.

No refunds. Due to the digital nature of this product, all sales are final. Please try the free trial before purchasing.
