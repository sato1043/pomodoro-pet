PRIVACY POLICY

Pomodoro Pet
Last Updated: 2026-02-23


1. INTRODUCTION

This Privacy Policy describes how Pomodoro Pet ("the Software"), developed
by sato1043@updater.cc ("the Developer"), handles your information. The
Developer is committed to protecting your privacy.


2. INFORMATION WE DO NOT COLLECT

The Software does NOT collect any of the following:

- Personal information (name, email address, phone number, etc.)
- Account credentials or login information
- Location data
- Device identifiers or hardware information
- Usage analytics or telemetry
- Advertising identifiers
- Cookies or tracking data


3. LOCALLY STORED DATA

The Software stores the following data exclusively on your local device.
This data never leaves your device and is not transmitted to the Developer
or any third party.

(a) Settings (stored in %APPDATA%/pomodoro-pet/settings.json on Windows):
    - Timer configuration (work/break/long-break duration, number of sets)
    - Sound configuration (preset, volume, mute state)
    - Background behavior (audio/notification preferences)
    - Weather display settings (weather type, time of day, cloud density)
    - Character emotion parameter (affinity value)

(b) Statistics (stored in %APPDATA%/pomodoro-pet/statistics.json on Windows):
    - Daily aggregated data: completed and aborted pomodoro cycles,
      number of work/break phases completed, total work/break time
    - Keyed by date (YYYY-MM-DD format)

(c) License data (stored in %APPDATA%/pomodoro-pet/settings.json):
    - Device ID (randomly generated UUID)
    - JWT token (registration status proof, contains device ID and
      key hint only)
    - Download key (if registered)

All data is stored as plain JSON files. No encryption is applied. No
personal information is included in these files.


4. SYSTEM NOTIFICATIONS

The Software uses the operating system's notification API to display local
notifications (e.g., when a pomodoro phase completes while the application
is in the background). These notifications are processed entirely on your
device and do not involve any network communication.


5. NETWORK COMMUNICATION

The Software communicates with the following external services:

(a) License verification server (GCP Cloud Function):
    - Sends: Device ID (randomly generated UUID, not personally
      identifiable), application version, download key (if registered)
    - Purpose: Trial period management, license verification, JWT
      token issuance
    - Frequency: Once per application launch (skipped if cached
      locally)

(b) GitHub (github.com):
    - Purpose: Internet connectivity check (HTTP HEAD request),
      application update check and binary download
    - No personal data is sent

The Software does NOT send:
    - IP addresses are visible to servers as part of standard HTTP
      communication, but are not collected or stored by the Developer
    - Hardware information, OS information, or usage analytics
    - Personal information of any kind


6. THIRD-PARTY SERVICES

The Software is distributed through itch.io. Your purchase, account
management, and payment processing on itch.io are governed by itch.io's
own privacy policy (https://itch.io/docs/legal/privacy-policy).
After installation, the Software communicates with the Developer's license
verification server and GitHub as described in Section 5. The Software does
not communicate with itch.io directly.


7. FUTURE CHANGES

The Developer may introduce the following features in future versions of
the Software:

- Analytics or crash reporting to improve software quality
- Cloud data synchronization to enable cross-device usage
- Other features that may involve network communication

If any such features are introduced, this Privacy Policy will be updated
before the release of the version containing those features. Users will
be notified of the changes through the application's update notes. Where
applicable, the Developer will obtain user consent before collecting or
transmitting any data.


8. DATA DELETION

All data stored by the Software resides on your local device. To delete
all data:

- Uninstall the Software, or
- Manually delete the %APPDATA%/pomodoro-pet/ directory

Note: Device registration data stored on the Developer's server
(device ID and registration status) cannot be deleted by the user.
Contact sato1043@updater.cc to request deletion of server-side data.

No data is retained by the Developer or any third party after deletion.


9. CHILDREN'S PRIVACY

The Software does not knowingly collect any personal information from
anyone, including children under the age of 13 (or the applicable age
in your jurisdiction). Since the Software collects no personal information,
no special provisions for children's data are necessary.


10. CONTACT

If you have any questions about this Privacy Policy, please contact:

    sato1043@updater.cc


11. GOVERNING LAW

This Privacy Policy shall be governed by and construed in accordance with
the laws of Japan.

1. はじめに

本プライバシーポリシーは、sato1043@updater.cc（以下「開発者」）が開発した
Pomodoro Pet（以下「本ソフトウェア」）における情報の取り扱いについて記述する。
開発者はユーザーのプライバシー保護に努める。


2. 収集しない情報

本ソフトウェアは以下の情報を一切収集しない。

- 個人情報（氏名、メールアドレス、電話番号等）
- アカウント認証情報やログイン情報
- 位置情報
- デバイス識別子やハードウェア情報
- 利用状況の分析データやテレメトリ
- 広告識別子
- Cookieやトラッキングデータ


3. ローカル保存データ

本ソフトウェアは以下のデータをユーザーのローカルデバイスにのみ保存する。
これらのデータはデバイスから外部に送信されることはなく、開発者や第三者に
転送されることもない。

(a) 設定（Windowsの場合 %APPDATA%/pomodoro-pet/settings.json に保存）:
    - タイマー設定（作業/休憩/長時間休憩の時間、セット数）
    - サウンド設定（プリセット、音量、ミュート状態）
    - バックグラウンド動作設定（音声/通知の有無）
    - 天気表示設定（天気タイプ、時間帯、雲量）
    - キャラクター感情パラメータ（親密度）

(b) 統計（Windowsの場合 %APPDATA%/pomodoro-pet/statistics.json に保存）:
    - 日次集計データ: 完了/中断したポモドーロサイクル数、
      作業/休憩フェーズの完了数、累計作業/休憩時間
    - 日付キー（YYYY-MM-DD形式）で管理

(c) ライセンスデータ（Windowsの場合 %APPDATA%/pomodoro-pet/settings.json に保存）:
    - デバイスID（ランダム生成UUID）
    - JWTトークン（登録状態の証明。デバイスIDとキーヒントのみ含む）
    - ダウンロードキー（登録済みの場合）

すべてのデータはプレーンなJSONファイルとして保存される。暗号化は適用されない。
これらのファイルに個人情報は含まれない。


4. システム通知

本ソフトウェアはOSの通知APIを使用してローカル通知を表示する（例: アプリが
バックグラウンドにあるときにポモドーロフェーズが完了した場合）。これらの通知は
デバイス上で完結し、ネットワーク通信を伴わない。


5. ネットワーク通信

本ソフトウェアは以下の外部サービスと通信する。

(a) ライセンス検証サーバー（GCP Cloud Function）:
    - 送信する情報: デバイスID（ランダム生成UUID、個人を特定しない）、
      アプリケーションバージョン、ダウンロードキー（登録済みの場合）
    - 目的: トライアル期間管理、ライセンス検証、JWTトークン発行
    - 頻度: アプリ起動ごとに1回（ローカルキャッシュがある場合はスキップ）

(b) GitHub（github.com）:
    - 目的: インターネット接続確認（HTTP HEADリクエスト）、
      アプリケーションのアップデート確認とバイナリダウンロード
    - 個人データは送信しない

本ソフトウェアが送信しない情報:
    - IPアドレスは標準的なHTTP通信の一部としてサーバーに見えるが、
      開発者はIPアドレスを収集・保存しない
    - ハードウェア情報、OS情報、利用状況の分析データ
    - いかなる個人情報も送信しない


6. 第三者サービス

本ソフトウェアはitch.ioを通じて配布される。itch.ioでの購入、アカウント管理、
決済処理はitch.io自身のプライバシーポリシー
（https://itch.io/docs/legal/privacy-policy）に準拠する。
インストール後、本ソフトウェアはセクション5に記載の通り、開発者のライセンス
検証サーバーおよびGitHubと通信する。本ソフトウェアはitch.ioとは直接通信しない。


7. 将来の変更

開発者は本ソフトウェアの将来のバージョンにおいて、以下の機能を導入する
可能性がある。

- ソフトウェア品質向上のためのアナリティクスまたはクラッシュレポート
- デバイス間利用を可能にするクラウドデータ同期
- その他ネットワーク通信を伴う可能性のある機能

これらの機能が導入される場合、当該機能を含むバージョンのリリース前に
本プライバシーポリシーを更新する。変更はアプリケーションのアップデートノートを
通じてユーザーに通知される。該当する場合、開発者はデータの収集や送信の前に
ユーザーの同意を取得する。


8. データの削除

本ソフトウェアが保存するすべてのデータはローカルデバイスに存在する。
すべてのデータを削除するには以下のいずれかを行う。

- 本ソフトウェアをアンインストールする
- %APPDATA%/pomodoro-pet/ ディレクトリを手動で削除する

注意: 開発者のサーバーに保存されたデバイス登録データ（デバイスIDおよび登録状態）
はユーザーが削除できない。サーバー側のデータ削除を希望する場合は
sato1043@updater.cc に連絡すること。

削除後、開発者や第三者にデータが残ることはない。


9. 子どものプライバシー

本ソフトウェアは13歳未満（または管轄地域で適用される年齢未満）の子どもを含め、
いかなる者からも個人情報を意図的に収集しない。本ソフトウェアは個人情報を収集
しないため、子どものデータに関する特別な措置は不要である。
